# goblog_study

## 目录结构

├── main          入口文件目录\
├── cache         缓存目录\
├── config        配置文件处理目录\
├── controller    控制器目录\
├── library       一些功能函数存放目录\
├── middleware    中间件存放目录\
├── model         数据库模型文件存放的目录\
├── provider      控制器和数据库交互读写的中间层处理函数目录\
├── public        对外公开的目录\
├── request       前端页面提交的数据模型定义目录\
├── route         路由器存放目录\
└── template      页面模板文件目录

mkdir main cache config controller library middleware model provider public request route template
